# Dashboard Interativo - Estatísticas FIFA 2026

Dashboard HTML interativo com estatísticas das seleções, ranking, comparativos e gráficos.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie o arquivo `index.html` para a raiz do repositório.
3. Vá em **Settings > Pages**.
4. Em **Build and deployment**, selecione:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Clique em **Save**.
6. Aguarde o GitHub gerar o link público.

O link ficará no formato:

`https://SEU_USUARIO.github.io/NOME_DO_REPOSITORIO/`
